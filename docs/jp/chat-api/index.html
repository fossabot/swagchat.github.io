Chat API

概要

Chat APIはチャットアプリケーションに必要なのルームやユーザの管理を行う為のREST APIです。 ステートレスなAPIなので水平スケールが可能です。

特定のデータベースやストレージに依存しないように設計されており現在は以下のプロダクトが利用可能です。 利用するプロダクトの設定はyamlによるファイルや環境変数、実行時パラメータで設定が可能です。設定できる内容は他にも数多くありますので詳細は設定をご参照下さい。

データベース

  • SQLite
  • MySQL
  • Google Cloud SQL

ストレージ

  • ローカルファイルシステム
  • Google Cloud Storage
  • Amazon S3

メッセージキュー

  • NSQ

プッシュ通知

  • Amazon SNS
